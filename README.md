# Sketch Selection Colors

一款帮助你快速查看和选择选区颜色的 Sketch 插件。

## ✨ 功能介绍

插件将显示应用于当前选区图层或描边填充的**纯色**和**渐变色**，并按颜色类型分组展示。支持自动检测颜色变量（Color Variables）。

### 主要特性

- 🎨 **纯色显示** - 显示色值/变量名、不透明度以及使用该颜色的图层数量
- 🌈 **渐变色显示** - 显示渐变类型（线性/径向/角度）和图层数量
- ✨ **颜色变量支持** - 自动检测并显示 Sketch 颜色变量名称
- 🔄 **自动刷新** - 面板跟随选区变化实时更新
- 👆 **悬停高亮** - 鼠标悬停时高亮显示

## 📖 使用方法

1. **选择图层** - 在 Sketch 中选择一个或多个图层
2. **运行插件** - 使用快捷键 `Ctrl + Shift + X` 或从菜单栏选择 *Plugins → Selection Colors → Show Colors*
3. **查看颜色** - 在弹出的面板中查看所有颜色
4. **选择图层** - 点击任意颜色行，然后点击 "Select Layers" 按钮即可选中所有使用该颜色的图层

## 📦 安装方法

1. 从 [Releases](https://github.com/Loooke19/Sketch-Selection-Colors/releases) 页面下载最新版本的 `.zip` 文件
2. 解压后双击 `.sketchplugin` 文件即可安装
